
# Projects Deep Learning - CNN & DNN  


Ce dépots contient deux projects de deep learning:  
    - Un project basé sur un **CNN (Convolution Neural Network)**  
    - Un project basé sur un **DNN (Deep Neural Network)**  
    - Le project Genetic Algorithm ne marche pas encore  

Chaque project est décliné en deux version :  
 - **Prototype**: version expérimental utilisée pour développé et améliorer le modèle  
 - **Final**: version stable et aboutie du modèle


## Structure du dépot


DNN:  
- Prototype: => Convolutional Neural Network/convolution  
- Stable: => Convolutional Neural Network/ConvDigit  

DNN:  
- Prototype: => Deep Neuron Network/deep_neuron_network  
- Stable: => Deep Neuron Network/AI-Digit (New Model)  
    
## Prérequis généraux

- Python 3.8+
- pip ou uv

Les dépendance sont décrites dans chaque project.

Se référer au README du project concerné :  
    - `CNN/README.md`  
    - `DNN/README.md`
